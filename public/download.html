宝塔面板安装脚本

2分钟装好面板，一键管理服务器

Linux面板 11.8.0

使用SSH连接服务器后执行：

CentOS
yum install -y wget && wget -O install.sh https://YOUR_DOMAIN.vercel.app/install/install_panel.sh && sh install.sh
Ubuntu/Debian
wget -O install.sh https://YOUR_DOMAIN.vercel.app/install/install_panel.sh && bash install.sh
一键更新
curl https://YOUR_DOMAIN.vercel.app/install/update6.sh|bash

注意：必须为没装过其它环境的新系统

Windows面板 8.5.2

1、使用官方安装程序安装，安装完先不要进入面板。

2、在cmd执行以下命令：

wget https://YOUR_DOMAIN.vercel.app/win/install/panel_update.py -O C:/update.py && "C:\Program Files\python\python.exe" C:/update.py 8.5.2

3、打开面板后绑定账号时输入任意账号密码即可。

aaPanel 7.0.25
Install
URL=https://YOUR_DOMAIN.vercel.app/install/install_7.0_en.sh && if [ -f /usr/bin/curl ];then curl -ksSO $URL ;else wget -O install_7.0_en.sh $URL;fi;bash install_7.0_en.sh
Upgrade
curl https://YOUR_DOMAIN.vercel.app/install/update_7.x_en.sh|bash
云安全监控 2.3.3

更新
curl https://YOUR_DOMAIN.vercel.app/install/update_btmonitor.sh|bash
